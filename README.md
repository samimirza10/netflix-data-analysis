# Netflix Data Analysis

## Overview

This project performs **Exploratory Data Analysis (EDA)** on the Netflix Titles dataset using Python. The goal is to clean the data, explore trends, and uncover insights about Netflix's catalog of movies and TV shows through data visualization.

---

## Dataset

The dataset contains information about Netflix titles, including:

- Movies and TV Shows
- Directors and Cast
- Country
- Release Year
- Date Added
- Rating
- Duration
- Genre

## Key Insights

- Movies significantly outnumber TV Shows in Netflix's catalog.
- TV-MA is the most common content rating.
- Netflix's content library grew rapidly after the early 2000s.
- The United States and India contribute the largest number of titles.
- International Movies and Dramas are among the most common genres.
- Most TV Shows consist of only one season.
- Most movies have a runtime between **80–120 minutes**.
